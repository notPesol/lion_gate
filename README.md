# lion_gate

## Used Libraries / Frameworks / Database
### frontend
* React
* React Router Dom
* Redux Toolkit
* Ant Design
* dayjs
* qrcode
### backend
* express
* mongoose
* cors
* bcrypt
* jsonwebtoken
* dotenv
## Used Database
* mongodb atlas

## How to run backend
* set .env file (DB_USERNAME, DB_PASSWORD, JWT_SECRET)
* cd ./backend
* npm install
* npm run dev

## How to run frontend
* cd ./frontend
* yarn
* yarn dev
